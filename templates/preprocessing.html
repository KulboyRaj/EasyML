Missing values occur when there are no data values in one or more columns of a dataset. Dealing with missing values is an essential preprocessing step, and it can be addressed by various techniques such as imputation or deletion.
?
Label Encoding is a technique used to convert categorical data into numerical format. It assigns a unique numerical value to each category, making it easier for machine learning algorithms to process the data.
?
Min-Max Scaling is a technique used to scale features to a range between 0 and 1. It helps in handling features with different scales and brings them to a common scale, which can improve the performance of certain machine learning algorithms.
?
Principal Component Analysis (PCA) is a dimensionality reduction technique used to reduce the number of features in a dataset while preserving most of its variance. It helps in simplifying the dataset and speeding up the computation in machine learning algorithms.
?
Log transformation is a technique to stabilize the variance of a dataset and make it more normally distributed. It involves replacing each data value x with its logarithm log(x).
?
Normalization is a technique used to scale numerical features to a standard range, typically between 0 and 1. It helps in bringing all features to a similar scale, which can improve the performance of certain machine learning algorithms, especially those sensitive to feature scales.
